# add-operation-audit-logs

## Why

后台管理系统需要记录关键 API 调用和管理员操作，方便追踪问题、审查敏感行为、定位权限变更来源。

## What Changes

- 新增 `audit_logs` 表记录 API 调用日志。
- 新增审计日志中间件，自动记录 `/api/*` 请求。
- 支持请求体截断和敏感字段脱敏。
- 支持 multipart 上传请求跳过 body 采集。
- 新增管理端审计日志查询接口。
- 支持 API 调用日志、登录日志、操作日志、权限变更日志、数据访问日志五类查询视图。

## Out of Scope

- 暂不做日志分表。
- 暂不做日志归档任务。
- 暂不做审计日志导出。
- 暂不做前端页面。
